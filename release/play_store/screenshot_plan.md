# Screenshot and Store Graphics Plan

## Purpose

This document defines the first Play Store visual asset plan for Bu mu, O mu?

Branch 5 is focused on publish readiness, so these assets should represent the existing app flow without adding new features.

## Required assets

- App icon
- Feature graphic
- Phone screenshots

## App icon

Use the Branch 4 custom launcher icon as the source direction.

For Play Store upload, prepare a high-resolution icon version if required by Play Console.

## Feature graphic

Recommended size: 1024 x 500

Recommended text: Bu mu, O mu?

Recommended subtext: Seç, keşfet, paylaş.

Visual direction:

- simple
- modern
- playful
- clean
- consistent with the app icon and app UI

## Phone screenshot plan

Recommended screenshot set:

1. Home screen
   Caption: Hızlı seçimlere başla

2. Quiz screen
   Caption: Bu mu, O mu? Soruları cevapla

3. Progress or mid-quiz screen
   Caption: Seçimlerinle ilerle

4. Result profile screen
   Caption: Eğlenceli sonucunu gör

5. Result actions screen
   Caption: Sonucunu paylaş

6. Replay state or result action screen
   Caption: Tekrar oyna, farklı sonucu keşfet

## Capture rules

- Use a real Android device or emulator.
- Prefer release build.
- Debug banner must not be visible.
- Text must not overflow.
- Result text must be readable.
- Status bar should look clean.
- Navigation bar is acceptable if present.
- Share sheet screenshot is not required.
- Store screenshots should focus on clean in-app screens.

## Review-critical represented flow

Home -> Quiz -> Result Profile -> Share / Replay / Home

## Reviewer notes

The app does not require login.

The app does not require payment.

The app does not need backend connectivity.

The quiz can be completed directly after opening the app.
